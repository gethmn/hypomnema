# Hypomnema

> *"a material memory of things read, heard, or thought"*
> — Foucault, on the hypomnemata of the ancient Greeks

A local daemon that indexes a Markdown directory and exposes search
(filesystem, content, semantic) and change events to any consumer —
most commonly an AI agent connected via MCP.
