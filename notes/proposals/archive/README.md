# Proposals — Archive

Frozen historical records of proposals that have been approved and whose canonical content has been distributed to the LDS layers under [`docs/`](../../../docs/).

These files are **not canonical**. They preserve:

- The reasoning at the time of approval (alternatives considered, scope debates)
- Content that didn't have a natural home in any LDS layer (personas, success metrics, journeys)
- A clean summary of "what we thought we were buying" if the feature is later revisited or rolled back

If an archived proposal disagrees with the LDS docs, the LDS docs win. The archive is read-only by convention.
